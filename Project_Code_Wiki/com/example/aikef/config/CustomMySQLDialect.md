# CustomMySQLDialect

## Class Profile
`CustomMySQLDialect` is a custom Hibernate dialect that extends `MySQLDialect`. Its primary purpose is to ensure that all tables created by Hibernate use the `utf8mb4` character set and `utf8mb4_0900_ai_ci` collation, which supports full Unicode (including emojis).

## Method Deep Dive

### `getTableTypeString()`
- **Overrides**: `MySQLDialect.getTableTypeString()`
- **Returns**: `" ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci"`
- **Effect**: Appends this string to `CREATE TABLE` statements generated by Hibernate.

## Usage Guide
Configured in `application.yml`:
```yaml
spring:
  jpa:
    database-platform: com.example.aikef.config.CustomMySQLDialect
```
